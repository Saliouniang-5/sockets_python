Projet Sockets en Python

 Description
Ce projet illustre l’utilisation des **sockets en Python** pour établir une communication entre un serveur et un client.  
Il permet de comprendre les bases de la programmation réseau : création de connexions, envoi et réception de messages, gestion des erreurs.

Fonctionnalités
- Serveur TCP qui écoute sur un port défini
- Client capable de se connecter au serveur
- Échange de messages entre serveur et client
- Gestion des connexions multiples (optionnel)
